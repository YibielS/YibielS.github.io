Program Details

An open-source resource management directory based on existing enterprise systems. Created for the purpose of initial training and simulation for IT students. Scope spans a light DB system written entirely in Java, as well as potential JDBC expansions. Users are able to modify objects, perform deletion, recovery, permissions management, and provisioning all through a simulated environment (CLI). Program is based on the Java language. Development is currently in progress.
