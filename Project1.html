Program Details

Program project in a Java environment, simulates conveyor package routing in warehousing/shipping environments. Based on a multi-threaded logic with an entirely scalable logic on a light codebase. Resolves and mitigates deadlocking by optimizing 'bottleneck' resource allocations.
